## Run

### <b> 1. Clone repo </b> <br/>
<pre>
$ git clone https://github.com/kechayias/chat-app.git
$ cd chat-app
</pre>
  
### <b> 2. Backend </b> <br/>
<pre>
$ cd server
$ npm install  
$ npm run dev 
</pre>

### <b> 3. Frontend </b> <br/>
<pre>
$ cd client 
$ npm install
$ npm start 
</pre>
